# Brainfuck

A Brainfuck interpreter written in elixir

## Running 
call `mix run --no-halt` to run the interpreter

## Example program
```bf
>++++++++[<+++++++++>-]<.>++++[<+++++++>-]<+.+++++++..+++.>>++++++[<+++++++>-]<++.------------.>++++++[<+++++++++>-]<+.<.+++.------.--------.>>>++++[<++++++++>-]<+.
```

Outputs: `Hello, World!`
